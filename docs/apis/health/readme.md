# Health APIs

## Overview

Fitness and wellness data for healthcare platforms and fitness applications.

## Endpoints

### [Fitness Exercises](./exercises.md) - ✅ MVP

Browse 1,500+ exercises with step-by-step instructions, target muscles,
secondary muscles, equipment requirements, and body part filters.

- **Status:** mvp
- **Endpoints:**
  - `GET /v1/health/exercises`
  - `GET /v1/health/exercises/{id}`
  - `GET /v1/health/exercises/body-parts`
  - `GET /v1/health/exercises/equipment`
  - `GET /v1/health/exercises/targets`
  - `GET /v1/health/exercises/search`
- **Credit Cost:** 1
