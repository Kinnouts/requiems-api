## Requiem API Overview

This folder contains developer-focused documentation for Requiem API.

Recommended entry points:

- `getting-started.md` – how to obtain an API key and make your first request.
- `architecture.md` – high-level architecture of the platform.
- `credits-pricing.md` – credit system, plans, rate limits, and pricing.
- `api-catalogue.md` – list of all APIs and their status (mirrors `apis.md`).
- `backend.md` – Go backend structure, modules, and patterns.
- `infra.md` – infrastructure and deployment reference (mirrors
  `infra/readme.md`).
